URL Lookalike Blocker

Options
⚠
Private browsing is not protected. This extension is not allowed to run in private windows, so it will not work there. To enable protection, open the Firefox menu (≡) → Extensions and Themes, find this extension, and set Run in Private Windows to Allow. Alternatively, type about:addons in a new tab's address bar.

Interface options

Whitelisted domains

Domains you have explicitly allowed. These will not be blocked even if they contain disallowed scripts.

Permitted languages

Select which languages are permitted beyond your locale defaults. Enabling a language allows its scripts in domain names and permits mixing those scripts within a single domain label.

This extension intercepts domain names that use characters from scripts not associated with your enabled languages, or that mix scripts in ways typical of lookalike spoofing attacks.

Latin-script languages are always permitted — see Help for the full list.
